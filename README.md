# Installations 

## To run the model
Inside the model folder do:
```
python3 -m venv venv
source venv/bin/activate
pip install ai-edge-model-explorer
pip install -e .
```

```
model-explorer --host=0.0.0.0 --extensions CFGgrid_ME 
```
